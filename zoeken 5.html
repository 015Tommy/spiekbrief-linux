Zoeken naar Commando's

Zoek naar Linux commando's en hun beschrijvingen

ls

Toont bestanden in de huidige map. Opties: -l (gedetailleerd), -a (alle bestanden inclusief verborgen).

Voorbeeld: ls -la - Toont alle bestanden inclusief verborgen in gedetailleerde lijst.
cd

Verandert naar een andere map. Bijv. cd /home gaat naar home, cd .. gaat omhoog.

Voorbeeld: cd /home/user - Ga naar de home map van gebruiker.
pwd

Toont de huidige werkmap (print working directory).

Voorbeeld: pwd - Toont de volledige pad naar de huidige directory.
cp

Kopieert bestanden/mappen. Gebruik -r voor recursief kopiëren van mappen.

Voorbeeld: cp bestand.txt backup.txt - Kopieer bestand.txt naar backup.txt.
mv

Verplaatst of hernoemt bestanden/mappen.

Voorbeeld: mv oud.txt nieuw.txt - Hernoem oud.txt naar nieuw.txt.
mkdir

Creëert nieuwe mappen. -p voor geneste mappen.

Voorbeeld: mkdir -p pad/naar/map - Creëer geneste mappen.
rm

Verwijdert bestanden/mappen. -r voor mappen, -f voor force.

Voorbeeld: rm -rf map/ - Verwijder map recursief zonder bevestiging.
uname

Toont systeem informatie. uname -a voor alle details.

Voorbeeld: uname -a - Toont alle systeem informatie.
whoami

Toont huidige gebruikersnaam.

Voorbeeld: whoami - Toont huidige gebruiker.
date

Toont huidige datum en tijd.

Voorbeeld: date - Toont huidige datum en tijd.
df

Toont schijfruimte gebruik. df -h voor human-readable format.

Voorbeeld: df -h - Toont schijfruimte in human-readable format.
free

Toont geheugen gebruik.

Voorbeeld: free -h - Toont geheugen gebruik in human-readable format.
ps

Toont lopende processen. ps aux voor alle processen.

Voorbeeld: ps aux - Toont alle lopende processen met details.
top

Interactieve proces viewer.

Voorbeeld: top - Start interactieve proces viewer.
kill

Stopt een proces. Gebruik PID nummer.

Voorbeeld: kill 1234 - Stopt proces met PID 1234.
grep

Zoekt naar patronen in tekst. Bijv. grep "zoekterm" bestand.txt zoekt naar "zoekterm" in bestand.txt. Opties: -i (case-insensitive), -r (recursief in mappen).

Voorbeeld: grep "error" log.txt - Zoek naar "error" in log.txt.
cat

Toont inhoud van bestanden. cat bestand.txt print de inhoud naar terminal.

Voorbeeld: cat bestand.txt - Toont inhoud van bestand.txt.
head

Toont eerste regels van bestand. head -n 10 bestand.txt toont eerste 10 regels.

Voorbeeld: head -n 5 bestand.txt - Toont eerste 5 regels van bestand.txt.
tail

Toont laatste regels van bestand. tail -n 10 bestand.txt toont laatste 10 regels.

Voorbeeld: tail -n 5 bestand.txt - Toont laatste 5 regels van bestand.txt.
rmdir

Verwijdert een lege map. Voor niet-lege mappen gebruik rm -r map_naam.

Voorbeeld: rmdir lege_map - Verwijder lege map.
touch

Creëert een leeg bestand of update de modificatietijd van een bestaand bestand.

Voorbeeld: touch nieuw_bestand.txt - Creëer een leeg bestand.
chmod

Verandert permissies van bestanden/mappen. Bijv. chmod 755 bestand geeft eigenaar read/write/execute, groep en anderen read/execute.

chown

Verandert eigenaar van bestanden. chown gebruiker bestand.

useradd

Creëert een nieuwe gebruiker. useradd nieuwe_gebruiker.

passwd

Verandert wachtwoord van gebruiker.

apt update

Update pakketlijsten.

Voorbeeld: apt update - Update pakketlijsten.
apt install

Installeer pakket. Bijv. apt install firefox.

Voorbeeld: apt install firefox - Installeer Firefox.
apt remove

Verwijder pakket.

Voorbeeld: apt remove firefox - Verwijder Firefox.
yum install

Installeer pakket op RPM-gebaseerde systemen.

Voorbeeld: yum install firefox - Installeer Firefox op RPM systeem.
bg

Zet proces op achtergrond.

Voorbeeld: bg - Zet gestopt proces op achtergrond.
fg

Breng proces naar voorgrond.

Voorbeeld: fg - Breng achtergrond proces naar voorgrond.
ifconfig

Toont netwerk interfaces (ouder commando).

Voorbeeld: ifconfig - Toont netwerk interfaces.
ip

Modern commando voor netwerkconfiguratie. ip addr show toont IP adressen.

Voorbeeld: ip addr show - Toont IP adressen.
ping

Test connectiviteit naar host. ping google.com.

Voorbeeld: ping google.com - Test connectiviteit naar google.com.
ssh

Verbind veilig naar remote server. ssh user@host.

Voorbeeld: ssh user@host - Verbind naar remote server.
wget

Download bestanden van internet. wget url.

Voorbeeld: wget https://example.com/file.zip - Download bestand.
